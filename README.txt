Project Name:
Hospitality Employee Dashboard

Tools Used:
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- KPI Cards

Objective:
Analyze employee distribution and salary expenditure across departments in a hospitality organization.

KPIs:
- Total Employees: 24
- Total Salary: RM 98,800
- Departments: 8
- Top Department: Kitchen

Key Insights:
- Kitchen has the highest salary expenditure (RM 32,100).
- Kitchen and F&B have the highest employee count (6 employees each).
- HR, Sales, Management, and Maintenance have only 1 employee each.

Skills Demonstrated:
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Business Reporting
