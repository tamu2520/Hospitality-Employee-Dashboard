# Hospitality Employee Dashboard

## Project Objective
Analyze employee data and create a dashboard for business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning

## Cleaning Steps
1. Removed duplicates
2. Standardized salary values
3. Cleaned the experience column
4. Fixed missing values

## KPIs

- Total Employees
- Average Salary
- Department Count
- Average Experience
